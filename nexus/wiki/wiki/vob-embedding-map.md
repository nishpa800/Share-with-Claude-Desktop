# VOB Embedding Map — Which Detection Plots Fire Inside VOB Zones

This is the central cross-reference for Anish's thesis: when a detection plot
fires INSIDE a Volume Order Block, it's a stronger signal than the same plot
firing in open air. This map categorizes every detection plot by its structural
relationship to VOB zones.

## Classification System

**EMBEDDED (E):** The detection plot's logic inherently references or occurs
within a VOB-like zone structure (EMA crossover zones, TNT zones, supertrend
levels). These are structurally tied to zones.

**COINCIDENT (C):** The detection plot fires independently but frequently
co-occurs with VOB zone formation/testing due to shared volume/momentum
conditions. Not structurally tied, but statistically correlated.

**INDEPENDENT (I):** The detection plot has no structural or statistical
relationship to VOB zones. Fires based on entirely different conditions.

**AMPLIFIER (A):** The detection plot AMPLIFIES VOB signals — when it fires
near a VOB zone, the zone's conviction increases (volume confirmation,
momentum confirmation, etc.).
